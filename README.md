# Flynncade

Yo. It's a VR arcade for Alloverse. Get it.

## Developing

Application sources are in `lua/`.

To start the app and connect it to an Alloplace for testing, run

```
./allo/assist run alloplace://nevyn.places.alloverse.com
```
